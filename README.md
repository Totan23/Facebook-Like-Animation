# Facebook Like Animation

Una aplicación iOS que replica la animación de reacciones de Facebook. Al mantener presionada la pantalla, aparece una barra animada con diferentes opciones de reacciones (me gusta, corazón, sorprendido, risa, tristeza y enojo) que responden con animaciones interactivas cuando el usuario desliza sobre ellas.
